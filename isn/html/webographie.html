Une liste de sites d'intérêt

  • W3school : LA référence de la création de site Web. Site en Anglais, très complet. Contient de nombreux tutoriels et toutes les références des langages utiles pour un site (html, css, javascript et associés, php, sql). La possibilité de tester les modifs d'un bout de code de tutoriel en direct est très appréciable !
  • Mozilla Developper Network (MDN) : Site très complet (en Français) pour développer des pages Web. Partie HTML, Partie CSS.
  • Validateur HTML et Validateur CSS : Service de validation de fichier HTML ou CSS. Indispensable pour s'assurer qu'on code bien dans les règles de l'art. Tout fichier doit passer par là avant sa publication.
  • Cours de Pierre Giraud : Un bon cours complet en français en vidéo avec l'essentiel en texte. Bien fait et utile.
  • Cours HTML.net : Un cours un peu vieillissant mais assez clair et qui présente des exemples.
  • Mémento CSS : Page extraite du site d'Open Classrooms listant les propriétés CSS principales.
  • Mémento éléments HTML : Page extraite du site d'Open Classrooms listant les éléments HTML principaux.
  • ISN & Math : Le site d'Olivier Lécluse : des cours de math + ISN du niveau lycée.
  • Spécialité ISN par F. Sincère : Site élaboré par un collègue de l'académie de Grenoble. Cours de HTML + CSS et Python, et d'autres choses encore...
  • Syntaxe des sélecteurs CSS : Présente de façon un peu austère la syntaxe possible des sélecteurs CSS. C'est ardu à lire, mais c'est très complet.
  • À propos de couleurs : Présente quelques infos sur les couleurs. Ce n'est pas d'un intérêt majeur mais ça peut aider à maîtriser l'utilisation des couleurs sur un site Web.
  • Jardin Zen CSS : Ce site présente la même page Web avec des fichiers CSS associés différents. Très impressionnant pour découvrir l'utilité du CSS.
  • Flexbox : La mise en place des différentes parties d'une page Web se fait aujourd'hui avec les propsriétés CCS de Flexbox. À découvrir sur ce site par exemple.
  • Caractères spéciaux : Listing des caractères spéciaux et de leur codage HTML ou ASCII.
  • Photos libres : Catalogue de photos et images libres de droit.
  • Tkinter : Documentation en Français sur tkinter, assez complète et pratique.
  • Python : Version en ligne du livre "Apprendre à programmer avec Python 3" de G. Swinnen. Intéressant et assez complet, mais long à lire...
