Logiciel Atom

J'ai découvert cet éditeur bien sympa au cours de mes divagations sur le net. Allez visiter la page officielle pour de belles infos complémentaires.
Ce qui me plait dans cet éditeur (outre son nom !) est la coloration syntaxique et surtout la complétion automatique qui m'autorise à oublier la syntaxe précise des langages et à quand même pouvoir coder assez rapidement.

Par ailleurs cet éditeur est modulaire et on peut ajouter plein de trucs en plus. Personellement, je n'ai rajouté qu'un "colorpicker" pour avoir une palette de couleur sous la main, mais je suis sûr qu'on peut trouver d'autres extensions intéressantes.
Voir ici la doc très succincte

Trucs ou astuces

Si la vue de l'arborescence des fichiers disparaît, on peut la remettre visible avec le menu
View -> Toggle Tree View

Dans le menu Edit -> Preferences -> Settings : cocher Soft Wrap pour que les lignes d'écritures ne dépassent pas la largeur de l'écran (ça facilite la lecture du code !).
